# NoteVault

Simple and modern note-taking web application with features such as CRUD, authorization and fully responsive UI.

## Technologies used:
    - React (Vite)
    - TypeScript
    - Tailwind CSS
    - React Router
    - Zustand
    - Firebase

## Usage

```bash
cd app
npm install
npm run dev
```
