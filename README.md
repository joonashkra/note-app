# Note / To-Do App

Web application with features such as CRUD, authorization and responsive UI.
Working on NodeJS / Express backend + refactoring frontend.

## Technologies used:

    - React (Vite)
    - TypeScript
    - Tailwind CSS
    - React Router
    - Zustand
    - Firebase (temporary)

## Usage

```bash
cd frontend
npm install
npm run dev
```
